ELECHOUSE RFID Serial Config Tool

Configure the ESP32-S3 LF/HF Network RFID reader from the browser through USB CDC or a product UART adapter. No ELECHOUSE account is required; commands are sent only to the serial port you select.

These are global device actions. Save stores runtime settings changed from any section of this page into device flash/NVS; Restart reboots the hardware.

This browser does not support Web Serial. Please use desktop Chrome, Edge, or another Chromium-based browser over HTTPS or localhost. Safari, Firefox, iOS, and most in-app browsers cannot open serial ports.
